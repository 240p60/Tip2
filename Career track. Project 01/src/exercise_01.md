### Колонка «2-3 года» — заполнена

**Карьерный статус:** Middle QA Engineer в продуктовой команде продуктовой IT-компании РФ (Ozon / Avito / Yandex / Сбер). Год работы junior + 1-1,5 года middle.

**Навыки и знания (целевой профиль через 2-3 года):**
- Python для автотестов (Pytest, фикстуры, моки).
- Playwright (UI) и Requests/HTTPX (API), уверенный опыт ≥1 год.
- SQL уровня Middle (CTE, оконные функции, EXPLAIN).
- CI/CD: GitLab CI / GitHub Actions — самостоятельная настройка пайплайна автотестов.
- Базовое нагрузочное тестирование (k6 / JMeter), 1-2 реальных проекта.
- Контейнеризация: Docker (запуск тестовых стендов локально).
- Английский — уверенный B2 (читаю/пишу/обсуждаю на стендапах).
- Jira/Confluence — уверенно, включая JQL.
- Soft skills: код-ревью тестов, peer-фидбек, ведение мини-фич end-to-end.

**Доступные ресурсы для развития (через 2-3 года):**
- Реальный продакшен и команда — главный источник опыта.
- ДМС и оплата конференций → 1-2 крупных QA-конференции в год (Heisenbug, QA Talks).
- Внутреннее QA-сообщество компании, гильдии, митапы.
- Менторская поддержка senior-инженеров.
- Свободные 5-7 часов в неделю на pet-проекты и обучение (English Speaking Club, чтение книг).

**Шаги, чтобы туда попасть (по результатам анализа вакансий):**
1. **Месяцы 0-3:** Подготовка по списку пробелов из §3 → pet-проект с Pytest + Playwright + GitHub Actions + Postman-коллекцией. Английский фоном.
2. **Месяцы 3-6:** Отклики на стажировки/junior вакансии (Яндекс, Avito, Сбер, Ozon). Параллельно — менторство через GetMentor.
3. **Месяцы 6-12:** Первая работа junior. Закрыть испытательный срок, выйти на самостоятельные таски. Подтянуть Docker, базовый CI, начать вникать в нагрузочное.
4. **Месяцы 12-24:** Промоушен на middle (внутри компании или сменой места). Самостоятельная настройка автотест-инфраструктуры на одном из сервисов. Защита миддл-грейда.
5. **Месяцы 24-36:** Стабильная работа middle. Углубление в одну из специализаций (security, perf, mobile) — на основе того, что окажется ближе. Первое выступление на внутреннем митапе. Подготовка к senior-грейду к концу 4-го года.
